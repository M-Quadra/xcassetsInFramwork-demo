# xcassetsInFramwork-demo

Using .xcassets in .framwork

曾经在StackOverflow看到的framework资源调用方法, 出于(qiáng)追(pò)求(zhèng)拿来尝试framework调用`.xcassets`。机缘巧合之下留了一篇博文记录。

>
前几天收到一封回复通知邮件, 我一看, 原来是有位朋友想问个demo。我想起来, 最近几个版本Xcode的设置有点不一样了。说时迟，那时懒。啪, 的一下, 怠惰上身, 很快啊, 看完邮件我就去烧夜之城了。  
一个开局闪退, 一个死机重启, 我全都忍了啊。  
我说750Ti 1G你不行, 性能不够用。它说可以试试。好吧, 试试就逝世。  
设置, 视频, 分辨率。720, 1080, 2k, 然后再也进不去了, 只留下了一个logo。  
我一看, 哦, 原来是显存爆了。手动改了本地设置, 终于救了回来。  
我劝这位750Ti 1G跑2k的穷逼, 耗子尾汁。不要搞拖延症, 这样好吗？这样不好。要讲code, 要赚钱还债, 快去干活!

既然最新版的Xcode稍微有些不同, 那就弄个demo, 顺便修缮。
